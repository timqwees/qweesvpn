# Реферальная система CoraVPN

## Обзор

Улучшенная реферальная система с гибкими настройками вознаграждений через ENV переменные.

## Логика работы

### Для реферала (тот, кто пригласил)
- **Получает:** Бонусные дни к подписке
- **Размер:** `%` от количества дней покупки приглашенного клиента
- **Начисление:** Автоматически при каждой покупке реферала

### Для приглашенного (тот, кто перешел по ссылке)
- **Получает:** Скидка на все тарифы
- **Размер:** Фиксированный процент от стоимости
- **Применение:** Единоразово при активации реферальной ссылки

## Настройка через ENV переменные

```bash
# Процент бонусных дней для реферала (по умолчанию: 10)
REFER_REFERRAL_DAYS_PERCENT=10

# Процент скидки для приглашенного пользователя (по умолчанию: 10)
REFER_INVITED_DISCOUNT_PERCENT=10
```

## Примеры расчетов

### Пример 1: Настройки по умолчанию
- REFER_REFERRAL_DAYS_PERCENT=10
- REFER_INVITED_DISCOUNT_PERCENT=10

**Сценарий:** Клиент покупает 30 дней за 300₽ по реферальной ссылке

**Результат:**
- Реферал получает: 30 дней × 10% = **3 бонусных дня**
- Приглашенный получает: скидка 10% на все тарифы

### Пример 2: Агрессивная маркетинговая кампания
- REFER_REFERRAL_DAYS_PERCENT=15
- REFER_INVITED_DISCOUNT_PERCENT=25

**Сценарий:** Клиент покупает 60 дней за 600₽ по реферальной ссылке

**Результат:**
- Реферал получает: 60 дней × 15% = **9 бонусных дней**
- Приглашенный получает: скидка 25% на все тарифы

## Структура классов

### `ReferConfig`
- Централизованное управление конфигурацией
- Валидация настроек
- Удобные методы для получения параметров

### `Refer`
- Управление реферальными ссылками
- Активация ссылок
- Применение скидок для приглашенных

### `Bonus`
- Начисление бонусных дней рефералам
- Детальное логирование операций
- Возврат результата операции

## Использование

### Активация реферальной ссылки
```php
$refer = new Refer();
$result = $refer->setRefer($userUniID, 'ABC123', '/profile');
```

### Начисление бонусов при покупке
```php
$bonus = new Bonus();
$result = $bonus->add_bonus_days($buyerUniID);
```

### Получение текущей конфигурации
```php
$config = ReferConfig::getAllConfig();
// [
//   'referral_days_percent' => 10,
//   'invited_discount_percent' => 10,
//   'invited_discount_multiplier' => 0.9
// ]
```

## Валидация конфигурации

```php
$errors = ReferConfig::validateConfig();
if (!empty($errors)) {
    // Обработка ошибок конфигурации
}
```

## Логирование

Система детально логирует все операции:
- Активация реферальных ссылок
- Начисление бонусных дней
- Ошибки и исключения

Формат логов:
```
[2024-01-01 12:00:00] [REFER_BONUS] Реферал user123 получил 3 бонусных дней от покупки клиента client456 (30 дней, 10%)
[2024-01-01 12:01:00] [SUCCESS] setRefer: Пользователь с uniID=client456 активировал реферальную ссылку 'ABC123' (рефер: uniID=user123). Скидка 10% применена.
```

## Безопасность

- Защита от повторной активации
